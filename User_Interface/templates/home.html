Drawbot

Accueilez Drawbot

Ce robot est capable de réaliser des dessins sur une feuille de papier qui est disposée au sol.
Il dispose d'un panel de divers modèles de dessins de base pré-installés.
Vous pourrez aussi le commander avec votre voix avec notre module de commande vocale.
Enfin, un mode capture y est intégré afin que, losrque vous lui présentez des images, il pourra les reproduire.

Vérifiez le modèle

N'hésitez pas à tester si le modèle que vous voulez dessiner est dans notre base de données.
